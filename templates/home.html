About the CRUD App

A web-based application to manage student information.

Key Features

  • Create: Easily add new student records using a web form.
  • Read: View a detailed list of all students stored in the database.
  • Update: Edit and update student information to keep records accurate.
  • Delete: Remove student entries when necessary.

Technology Stack

  • Flask: A lightweight Python web framework for backend logic and routing.
  • SQLite: A serverless database for storing student data securely.
  • HTML and CSS: Used to design the structure and style of the web app.
  • W3.CSS: A modern CSS framework integrated to enhance styling and responsiveness.
  • VS Code: The integrated development environment (IDE) used for writing and managing the code.

How to Use the App

  1. Use the navigation bar at the top of the page to navigate through the app.
  2. Click on the Add Student tab to input new student information into the database.
  3. Fill out the form with the required details and submit it to add the student.
  4. To view or manage existing students, click on the Review and Edit Students tab.
